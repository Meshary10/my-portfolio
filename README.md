# Personal Portfolio Website

A clean, responsive portfolio website for Meshari Alyami — CloudOps & MLOps Engineer. Built with vanilla HTML and CSS, no frameworks or build tools required.

## Features

- **Single-page design** with smooth-scrolling navigation (About, Skills, Projects, Contact)
- **Responsive layout** optimized for desktop, tablet, and mobile screens
- **Skills showcase** covering Python, Docker & Kubernetes, MLOps, Cloud Computing, Git, and Linux
- **Project gallery** highlighting featured work including sentiment analysis, data science, Terraform IaC, and MLOps pipelines
- **Contact section** with email, phone, and LinkedIn links

## Tech Stack

- HTML5 & CSS3 (no JavaScript)
- [Font Awesome 6.1.1](https://fontawesome.com/) for icons (via CDN)
- [Google Fonts — Poppins](https://fonts.google.com/specimen/Poppins) for typography (via CDN)

## Project Structure

```
├── src/
│   ├── index.html    # Main page
│   └── style.css     # All styles
├── images/           # Project screenshots and background images
└── README.md
```

## Getting Started

No build step needed — just open the HTML file in a browser:

```sh
open src/index.html
```

To deploy, copy the `src/` and `images/` directories to any static hosting provider (GitHub Pages, Netlify, Vercel, etc.).
